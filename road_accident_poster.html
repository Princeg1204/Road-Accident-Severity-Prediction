Problem Statement

Introduction & Objective

With rapid urbanization, vehicle density has skyrocketed — making accident prediction a critical public-safety challenge. Historical accident data contains rich patterns that ML can surface: temporal spikes, weather correlations, and geographic hotspots remain invisible to traditional analysis.

The US Accidents dataset captures 100,000 real incidents from California, Ohio, and surrounding states between 2016–2020. Each record includes severity labels (1=Minimal → 4=Critical), alongside meteorological readings, road geometry, and time-of-day context.

Core Objectives

① Classify accident severity across 4 classes   ② Identify high-risk feature combinations   ③ Compare baseline vs. tuned model performance   ④ Provide actionable insights for traffic authorities

Pipeline

Methodology

01
Data Ingestion
Loaded US Accidents CSV via pandas chunking (100K records). Dataset spans 46 columns: GPS coords, weather readings, road features, and severity labels.
02
Preprocessing
Imputed missing values (mean for numerics, "Unknown" for weather). Dropped irrelevant columns (IDs, URLs). LabelEncoded Weather_Condition into numeric classes.
03
Feature Selection
Retained 8 predictive features: Temperature(F), Humidity(%), Pressure(in), Visibility(mi), Wind_Speed(mph), Weather_Condition, Start_Lat, Start_Lng.
04
Train / Test Split
80/20 stratified split: 79,126 training samples · 19,782 test samples. Target: Severity (4-class multiclass problem).
05
Model Training & Tuning
Trained Logistic Regression, Decision Tree, Random Forest. Hyperparameter tuned Decision Tree via GridSearchCV (3-fold CV). Best params: max_depth=None, min_samples_leaf=4, min_samples_split=10.
06
Evaluation
Assessed via Accuracy, Precision, Recall, F1-Score, and Confusion Matrix. EDA visualizations guided feature importance understanding.

Impact

Conclusions & Applications

The tuned Decision Tree outperforms all baselines, achieving 90.83% accuracy on unseen data through GridSearchCV optimization. Logistic Regression struggled with the non-linear severity boundaries (56.2%), confirming this is a complex classification problem.

Temperature and geographic coordinates emerged as the strongest predictors — suggesting that location-specific climatic conditions create consistent accident severity signatures.
